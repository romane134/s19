                  PETIT BAIN: TUTORS INFORMATION              
*****************************************************************

Welcome to the Petit Bain, a coding initiation module created by 19 students

This project has 3 independent modules: C, Shell, Web

 --------------------------------------------------------
|			MODULE C                         |
 --------------------------------------------------------

HOW TO INSTALL:

1. Type command "cd /Users/event/Desktop/event/petit_bain/module_c"
2. Type command "./install.sh en". en is a language parameter, 
   avialable languages are en / fr / nl
3. Close terminal window and open a new one. IMPORTANT: This module 
   wont work in the same window where it was installed.

HOW TO LAUNCH / PLAY:

1. Open a new terminal window (fullscreen is highly recommended)
2. Type command "cd /Users/event/Desktop/event/petit_bain/module_c" 
3. Type "howtoplay" and enter.
4. Read the instructions. :)

HOW TO RESET:

If a new user wants to restart the game or there is a new user:

1. Type "resetgame" and hit enter.
2. Confirm by writing "Y" and enter. All progression will be lost.

HOW TO CHANGE LANGUAGE:

1. Type "setlanguage"
2. Choose your language (write en / fr / nl and hit enter)

 --------------------------------------------------------
|			MODULE SHELL                     |
 --------------------------------------------------------

1. Open a new terminal window (fullscreen is highly recommended)
2. Type command "cd /Users/event/Desktop/event/petit_bain/module_shell/ShellGame"
3. Type the command "source script.sh fr" or "source script.sh en"
4. Module will launch automaticly

HOW TO LAUNCH / PLAY:

1. Open a new terminal window
2. Type command "cd /Users/event/Desktop/event/petit_bain/module_shell/ShellGame"
3. Type command "sh delete.sh"
4. Type the command "source script.sh fr" or "source script.sh en"

HOW TO CHANGE LANGUAGE:

1. Follow the steps below to LAUNCH / PLAY choosing the desired language in step 4.
