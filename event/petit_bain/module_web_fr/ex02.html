Harder, better, stronger!

Bon, il est temps maintenant de sortir de google chrome, et de créer/modifier directement vos fichiers HTML et CSS...



Ouvrez le dossier petit_bain_web qui se trouve sur le Bureau de votre poste de travail.
Vous y trouverez les fichiers 19.html et 19.css, et quelques images en plus.

Ouvrez ces fichiers 19.html et 19.css avec Sublim Text, en glissant le dossier petit_bain_web sur cette icône dans les applications.
(Sublim Text est un éditeur de texte, comme Word ou Open Office, mais spécifiquement pour le code. Il en existe d'autres, mais pourquoi pas lui ?)


Vous pouvez maintenant écrire du code html dans 19.html, à l'endroit indiqué, et du css dans 19.css
Pour vous faciliter la tâche si vous voulez inclure des images, nous vous avons déjà fourni quelques unes.


Pour visualiser votre travail, rien de plus simple : double-cliquez sur 19.html, le fichier va s'ouvrir dans un navigateur web.
À chaque fois que vous modifiez l'un des fichiers 19.html ou 19.css, vous n'aurez qu'à rafraichir la page dans le navigateur.


Une astuce : sur votre page web dans le navigateur : click droit, puis 'Inspect'
La console du navigateur s'ouvrira.
Et vous pourrez y voir tout votre code et le modifier !
Vous verrez en direct les modifications sur votre page, ce qui est très pratique pour travailler.
Mais attention, ces modifications sont provisoires et disparaîtront au prochain rafraichissement de page.
N'oubliez pas de copier-coller ces modifications provisoires dans vos fichiers 19.html et 19.css pour les enregister.


Vous avez carte blanche, soyez créatif.ve.s, faites-vous plaisir !
