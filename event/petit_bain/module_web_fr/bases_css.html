Essayons !

Observez bien le code et le résultat.

Puis, essayez par exemple :
de changer la couleur du texte
de changer la couleur de fond du titre
de souligner seulement la 2ème phrase
de centrer le texte
de remettre Archimède à l'endroit (voir la propriété 'transform')

À chaque changement que vous ferez dans le code, il vous faudra cliquer sur la flèche 'refresh' pour voir le résultat

Attention, si vous rafraîchissez la page, toutes vos modifications dans le code disparaîtrons.
